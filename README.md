# Installation (Linux)

__Start__
git clone https://github.com/Dersw/IvanovK_cmake_multiproject.git
cd IvanovK_cmake_multiproject

cd projects && mkdir lib && cd lib
git clone https://github.com/google/googletest.git
cd ../..

__Build__
chmod +x build.sh
./build.sh

__Run Tests__
cd build_ninja\
./run_tests.sh
